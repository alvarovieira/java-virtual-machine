void menu ( )

Shows the JVM main menu.

◆ viewer()

void viewer ( )

Shows the .class viewer menu.

Shows the Classfile Viewer menu after the user chooses the option in the main menu. Here, the user can choose to set a .class file and after this, the class file structure is shown in this menu. The user also has the option to go to the JVM execution menu through the second option, if the .class file is already setted, the JVM loads the same file. The 3rd option is to go back to the previous menu.
